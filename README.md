# Python Weather CLI

Command line weather application that retrieves current weather data using the OpenWeather API.

## Features

- fetch weather by city
- display temperature and weather condition
- simple command line interface

## Technologies

- Python
- requests
- OpenWeather API

  ## Example

```
Enter city name: Berlin

Weather in Berlin
Temperature: 12 °C
Condition: clear sky
```

## Installation

Install required library:

```
pip install requests
```
